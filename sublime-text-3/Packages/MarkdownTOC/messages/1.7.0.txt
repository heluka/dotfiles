MarkdownTOC - 1.7.0

CHANGES

- Add 'style' feature


You can control the type of list representing the TOC:

```
<!-- MarkdownTOC style=unordered -->

- foo
  - bar
  - buz
- qux

<!-- /MarkdownTOC -->
```
```
<!-- MarkdownTOC style=ordered -->

1. foo
  1. bar
    1. qux
    2. quux
  2. buz
2. qux

<!-- /MarkdownTOC -->
```
